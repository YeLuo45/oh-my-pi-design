# oh-my-pi Design

Design documentation for **[oh-my-pi (omp)](https://github.com/can1357/oh-my-pi)** — Can Boluk's fork of pi-mono with a Rust core, LSP/DAP integration, 40+ LLM providers, and 32 built-in tools.

## What's inside

- 18 documentation pages covering all 3 tiers (Rust core, TypeScript middle, React web)
- VitePress 1.6.4 (Markdown → static site, deployed via GitHub Pages)
- Architecture diagrams, capability matrices, and isolation primitive deep-dives

## Quick start

```bash
npm install
npm run docs:dev      # local dev server
npm run docs:build    # production build
```
